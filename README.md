# Journey Solutions — Website UI Kit

## Overview
High-fidelity interactive prototype of the Journey Solutions app-selling website.

## Design Decisions
- Single warm off-white (#FAF7F2) background throughout
- Side navigation (not hamburger) with round icon buttons + visible labels
- Personal, story-driven copy style
- Oswald (display/UI) + Lora (body) type pairing
- Deep burgundy + warm amber brand palette

## Pages Prototyped
1. **Main** — Hero with personal intro, featured apps strip
2. **About Me** — Developer's personal story
3. **Apps** — Full app catalog grid
4. **App Detail** — Individual app with instructions + feedback form
5. **Contact** — Contact form

## Components
- `SideNav.jsx` — Side navigation with round icon buttons
- `Header.jsx` — Page header / hero section
- `AppCard.jsx` — App listing card component
- `HeroSection.jsx` — Landing hero
- `AboutSection.jsx` — About Me page
- `AppDetail.jsx` — App detail page
- `ContactForm.jsx` — Contact / feedback form

## Fonts
- Oswald (Bold 700, SemiBold 600, Medium 500) — Google Fonts
- Lora (Regular 400, SemiBold 600, Italic) — Google Fonts

## Icons
- Lucide Icons via CDN

## Notes
- This is a click-through prototype; forms are non-functional
- Navigation state is tracked client-side
